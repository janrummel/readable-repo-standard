# GLOSSARY — tidymark

- **tidy** — the top-level operation: apply all normalizers to a file in place.
- **normalize** — a single, pure transformation (whitespace, bullets, or headings).
- **bullet** — a Markdown list marker (`*`, `+`, `-`); tidymark standardizes on `-`.
- **heading level** — the number of leading `#` on a Markdown heading (`#` = level 1).
- **parked** — deliberately deferred with a recorded reason; not the next action.
